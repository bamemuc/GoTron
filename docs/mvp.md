# MVP Scope (2-3 Wochen)

## In Scope

- 2 Spieler
- 1 laufendes Match
- Feste Tickrate
- Richtungseingaben vom Client
- Kollisionen und Match-Ende
- Server in Docker, Client lokal

## Out of Scope

- 3-4 Spieler als Pflichtziel
- Matchmaking/Lobby-System
- Reconnect
- Accounts, Ranking, Persistenz
- Performance-Tuning vor funktionierendem End-to-End-Flow
